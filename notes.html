    Introduction to CSS
  • <head>
  • <link>
  • links the html file to .css file
  • CSS
  • language for styling HTML files
  • selector
  • points to the HTML you want to style
  • property
  • stylistic feature you want to change
  • value
  • how you want to change its features
  • common CSS properties
  • font-size
  • used to size the text of a tag
  • line-height
  • used to size the height of text lines
  • color
  • used to color the text

CSS notes 2
  • text align-CSS property
  • specifies the horizontal alignment of text
  • value:left, aligns the text left
  • value: right, aligns the text right
  • value: center, put text in center
  • text-decoration
  • adds underline, over-line, line-through, to chosen text
  • value: underline, horizontal line underneath the text
  • value: overline, horizontal line above text
  • value: line-through, line through the middle of a text
  • font-weight
  • bolds or light text
  • font-family
  • specifies the font for an element
  • background-color
  • sets the background clor of an element
  • class (attribute)
  • can use the same classes on multiple HTML elements
  • specifies one or more classes of a HTML element
